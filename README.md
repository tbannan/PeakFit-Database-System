# PeakFit Database Project

Final CS 411 database project for PeakFit Performance Center using MySQL and MongoDB.

## Project Overview
This project models the operations of a fitness and wellness organization. The database was designed to manage members, trainers, memberships, classes, workout plans, training sessions, payments, and equipment records.

## Project Contents
- MySQL relational database schema
- SQL table creation scripts
- Sample data insertion scripts
- SQL analysis and verification queries
- MongoDB collections and JSON documents
- Project presentation materials

## Technologies Used
- MySQL
- MongoDB Atlas
- SQL
- JSON
